# Butik Pastane ve Menü Yönetim Sistemi

[cite_start]Bu proje, Sistem Analizi ve Tasarımı dersi kapsamında geliştirilmiş web tabanlı bir CRUD uygulamasıdır[cite: 4]. Bir butik pastanenin ürün envanterini, fiyatlandırmasını ve kategorilerini dijital ortamda yönetmesini sağlar.

## Kullanılan Teknolojiler

- [cite_start]**Frontend:** Vanilla Javascript, HTML5, CSS3 (Single Page Application mimarisi) [cite: 12, 23]
- [cite_start]**Backend:** Node.js, Express.js [cite: 13]
- [cite_start]**Veritabanı:** SQLite [cite: 14]
- [cite_start]**API Dokümantasyonu:** Swagger UI [cite: 40]

## Temel Özellikler

- [cite_start]Menü ürünleri için tam CRUD (Oluştur, Oku, Güncelle, Sil) operasyonları [cite: 19]
- [cite_start]Asenkron API çağrıları (Fetch API) ile sayfa yenilenmeden çalışan dinamik arayüz [cite: 24, 25]
- [cite_start]İş mantığının (Business Logic) route'lardan bağımsız katmanlarda yönetilmesi [cite: 30]
- [cite_start]Hem istemci (Frontend) hem de sunucu (Backend) tarafında veri doğrulama (Validation) [cite: 28]
